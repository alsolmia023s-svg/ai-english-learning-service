# 2. ユーザージャーニーマップ

## ステージ構成
1. **認知・興味 (Awareness)**: 課題を感じ、サービスを知る
2. **利用開始 (Onboarding)**: アカウント作成、初回利用
3. **学習・体験 (Core Experience)**: 英文入力、AI添削、フィードバック確認
4. **継続・定着 (Retention)**: 復習、習慣化

## ジャーニー詳細

| ステージ | ユーザーの行動 | 思考・感情 | 接点・機能 |
| --- | --- | --- | --- |
| **1. 認知・興味** | ・仕事で英語メールを書くのに時間がかかっている<br>・「英語 メール 添削 AI」などで検索 | ・「もっと早く書きたい」<br>・「自然な表現を知りたい」<br>・「手軽に試したい」 | ・LP (ランディングページ)<br>・SNS広告 |
| **2. 利用開始** | ・LPを見て「これなら使えそう」と思う<br>・「無料で試す」ボタンを押す<br>・Googleアカウントでログイン | ・「登録が面倒だと嫌だな」<br>・「すぐに使えるかな？」 | ・ログイン画面 (NextAuth)<br>・ダッシュボード |
| **3. 学習・体験** | ・添削したい英文を入力フォームに貼り付け<br>・「添削する」ボタンをクリック<br>・数秒で結果が表示される<br>・修正点と解説を読む | ・「お、速い！」<br>・「なるほど、こう言えばいいのか」<br>・「解説が丁寧で分かりやすい」 | ・英文入力フォーム<br>・AI処理 (Claude Haiku 4.5)<br>・添削結果表示画面 (Diff表示) |
| **4. 継続・定着** | ・過去の添削履歴を見返す<br>・次回のメール作成時に学んだ表現を使う<br>・定期的にサービスを利用する | ・「前よりスムーズに書けるようになった」<br>・「履歴が残るから便利」<br>・「また使おう」 | ・履歴一覧画面<br>・お気に入り/保存機能<br>・メール通知 (Nodemailer) |

## 重要なモーメント
- **Aha! Moment**: 初回のAI添削結果が表示され、自分の英文がより自然で洗練されたものに変わった瞬間。解説によって「なぜ」修正されたかが腹落ちすること。
