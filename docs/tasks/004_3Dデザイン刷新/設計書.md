# 3Dリッチデザイン刷新 設計書

## 1. デザインコンセプト
「AI × 英語学習」の先進性と未来感を感じさせる、没入感のある3Dリッチなデザインを目指します。

### キーワード
- **Futuristic (未来的)**: 最先端のAI技術を表現
- **Immersive (没入感)**: 3D要素と奥行きのあるレイアウト
- **Premium (上質)**: 洗練されたタイポグラフィと配色

## 2. 技術選定

### 3Dライブラリ
- **React Three Fiber (R3F)**: ReactでThree.jsを扱うための標準的なライブラリ
- **Drei**: R3Fのヘルパーライブラリ（カメラコントロール、環境光、3Dテキストなど）

### アニメーション
- **Framer Motion**: 滑らかなページ遷移、スクロールアニメーション、要素の出現アニメーション

### スタイリング
- **MUI (既存)** + **Custom CSS/Emotion**: 既存のMUIをベースにしつつ、Glassmorphism（すりガラス効果）やグラデーションを多用したカスタムスタイルを適用

## 3. 実装内容

### 3.1 ヒーローセクション (Top Page)
- **3Dオブジェクト**: 回転する地球儀、または抽象的な幾何学オブジェクト（AIネットワークをイメージ）を配置
- **インタラクション**: マウスの動きに合わせて視点が微妙に変化、またはオブジェクトが反応する
- **背景**: 深みのあるダークグラデーション、パーティクル（粒子）アニメーション

### 3.2 UIデザイン刷新
- **Glassmorphism**: ヘッダー、カード、モーダルに「すりガラス効果」を適用し、奥行きを出す
- **Typography**: 見出しにモダンなフォント（Google Fonts: 'Inter' or 'Outfit'）を適用し、ウェイトや字間を調整
- **Color Palette**:
  - Base: Deep Navy / Black (`#0a192f`)
  - Accent: Neon Blue (`#64ffda`), Purple (`#bd34fe`)
  - Text: White / Light Gray

### 3.3 アニメーション
- スクロールに合わせて要素がふわっと浮き上がる (Fade In Up)
- ボタンのホバー時に光るエフェクト (Glow Effect)

## 4. ページ別変更点

### Landing Page (`/`)
- ヒーローセクションを3Dキャンバスに置き換え
- 特徴セクションのカードをGlassmorphismデザインに変更

### Auth Pages (`/auth/*`)
- 背景に動的な3Dパーティクルまたはグラデーションアニメーションを追加
- フォームコンテナをGlassmorphism化

### Dashboard / Admin (`/dashboard`, `/admin`)
- 既存の機能性を維持しつつ、ヘッダーやカードのデザインを統一感のあるものにアップデート

## 5. 導入パッケージ
```bash
npm install three @types/three @react-three/fiber @react-three/drei framer-motion
```
