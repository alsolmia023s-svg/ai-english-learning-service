# オーナー管理画面 動作確認チェックリスト

## 1. アクセス制御確認
- [ ] **管理者アクセス**
  - 管理者権限を持つユーザーでログインし、`/admin` にアクセスできること。
- [ ] **一般ユーザーアクセス制限**
  - 一般ユーザー（または未ログイン）で `/admin` にアクセスすると、ログイン画面または403エラーになること（今回はログイン画面へのリダイレクト実装）。

## 2. ダッシュボード表示確認 (`/admin`)
- [ ] **統計情報表示**
  - 「総ユーザー数」が表示されること。
  - 「総お問合せ数」が表示されること。

## 3. ユーザー管理確認 (`/admin/users`)
- [ ] **一覧表示**
  - ユーザー一覧が表示されること（名前、メール、権限、登録日）。
- [ ] **権限変更**
  - 「Promote/Demote」ボタンで権限を変更できること。
  - 変更後、一覧の「Role」が更新されること。
- [ ] **ユーザー削除**
  - 「Delete」ボタンでユーザーを削除できること。
  - 削除確認ダイアログが表示され、OKを押すと削除されること。

## 4. お問合せ管理確認 (`/admin/contacts`)
- [ ] **一覧表示**
  - お問合せ一覧が表示されること（名前、メール、件名、日時）。
- [ ] **詳細表示**
  - 「View」ボタンでお問合せ詳細ダイアログが表示されること。
  - 本文が正しく表示されること。
- [ ] **お問合せ削除**
  - 「Delete」ボタンでお問合せを削除できること。

## 5. ログアウト確認
- [ ] **ログアウト**
  - サイドバーの「ログアウト」ボタンでログアウトできること。
