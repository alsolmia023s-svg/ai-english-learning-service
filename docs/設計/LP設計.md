# LP設計書

## 1. ページ構成

### 1.1 ヒーローセクション
- **キャッチコピー**: 「あなたの英語を、もっと自然に、もっとプロフェッショナルに。」
- **サブコピー**: AIが瞬時に英文を添削・フィードバック。ビジネスメールもアカデミックなエッセイも、自信を持って発信しよう。
- **CTAボタン**: 「無料で始める」 (ログインページへ遷移)
- **ビジュアル**: シンプルかつ洗練された、ライティングやコミュニケーションを想起させる抽象的なイメージ、または実際の添削画面のモックアップ。

### 1.2 サービスの特徴 (Features)
3つの主要な特徴をアイコン付きで紹介。
1.  **AIによる瞬時添削**: Claude Haiku 4.5を使用し、待ち時間なく高精度なフィードバックを提供。
2.  **詳細な解説**: なぜ修正されたのか、理由を丁寧に解説。ただ直すだけでなく、学びがある。
3.  **履歴管理**: 過去の添削内容を保存。自分の成長を振り返ることができる。

### 1.3 使い方 (How it works)
3ステップで利用の流れを説明。
1.  **入力**: 添削したい英文を入力フォームに貼り付け。
2.  **添削**: ワンクリックでAIが解析・修正。
3.  **学習**: 解説を読んで納得。修正案をコピーして利用。

### 1.4 ボトムCTA
- **メッセージ**: 「まずは無料で体験してみませんか？」
- **ボタン**: 「今すぐ登録する」

### 1.5 フッター
- 著作権表示
- 利用規約、プライバシーポリシーへのリンク (今回はプレースホルダー)

## 2. デザイン要件 (UI/UX)
- **カラーパレット**:
    - Primary: 知的で信頼感のあるブルー系 (#1976d2)
    - Background: 清潔感のある白、またはダークモード時は目に優しいダークグレー
- **タイポグラフィ**: 読みやすさを重視したサンセリフ体 (Roboto, Noto Sans JP)
- **レスポンシブ**: モバイル、タブレット、デスクトップで最適に表示。
- **ダークモード**: システム設定またはユーザー切り替えに対応。

## 3. 技術実装方針
- **Framework**: Next.js 15 (App Router)
- **UI Library**: MUI (Material UI) v6
- **Icons**: MUI Icons (Material Icons)
- **Layout**: `Grid` または `Stack` コンポーネントを使用してレイアウト。
- **Theme**: `createTheme` を使用してカスタムテーマを定義し、`ThemeProvider` でラップする。
